# Rainydays

<img src="RainyDays.png" width=100%></img>

An e-commerce website for a fictitional outdoor clothing vendor as a course assignment for Front-End Development course at Noroff.

## Website

[Rainydays website](https://peppy-froyo-40c9b0.netlify.app)

## Description

The project contains:

- Home page
- About page
- Contact page
- Male jackets / Female jackets pages 
- Details page for the merchandise

## Tech Stack

The project was built with:

- HTML 5
- CSS
- Javascript

## Getting started

### Installing

1. Clone the repo:

`git clone git@github.com:EkaterinaNattrass/rainydays-website.git`

### Running

The project contains only static files, you can find the index page at `public/index.html`

### Contributing

This is an educational portfolio project, therefore contributions are not really applicable but feedback is always welcome.
